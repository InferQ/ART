# ART Framework Website

This is the marketing and documentation website for the ART (Agentic Reactive Triad) Framework.

## Tech Stack

- **React 19**
- **Vite**
- **Tailwind CSS**
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **React Router 7**

## Key Sections

- **PES Agent Deep Dive**: Visual explanation of the flagship orchestration engine.
- **The Reactive Triad**: Highlights the core pillars (Agent, State, UI).
- **Universal Coverage**: Shows supported LLM providers and integrations.
- **Technical Architecture**: Layered breakdown of the framework.
- **Interactive Features**: Dynamic release notes and capability grids.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Structure

- `/src/components`: UI components organized by section.
- `/src/lib`: Shared utilities and constants.
- `/src/docs-content.ts`: Embedded documentation content.

## Deployment

The site is built as a static application and can be hosted on GitHub Pages or any static hosting provider.
